# Análise de Dados de Veículos

Este projeto é um aplicativo simples desenvolvido com **Streamlit** para
análise exploratória de dados de anúncios de venda de veículos.

O app permite visualizar: Um **histograma** da quilometragem dos
veículos e um **gráfico de dispersão** entre quilometragem e preço

## Funcionalidades

-   Leitura de dados a partir de um arquivo CSV
-   Geração de gráficos interativos com Plotly
-   Interface web simples com botões interativos

## Estrutura dos Dados

O aplicativo utiliza o arquivo `vehicles_us.csv`

## Tecnologias Utilizadas

-   Python
-   Pandas
-   Plotly Express
-   Streamlit
